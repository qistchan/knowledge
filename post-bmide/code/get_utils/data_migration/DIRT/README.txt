DIRT Tool ready for svn.

Oracle connection information needed. Update oraenv.sh and tnsnames.ora (optional)
Must run on a Linux/Unix machine (tested on tnsp01974 )
Script connect to DB of choice (tested connection to DM1)
Following directory structure needed at any level. Set DIRT_ROOT in dirt_new.sh.

     DIRT
	SQL
	OUTPUT

SQL directory can contain any # of SQL (end with ; and extension sql). 
Output will be created in OUTPUT folder with same filename (txt extension)
